MISSION

สำหรับส่วนนี้จะทำงานแบบนี้
1. คิด Mission สำหรับการทดสอบ crypto ขึ้นมา ให้คุณไปค้นหาจากเนตหรืออื่นๆ มาประกอบ
2. Mission อาจจะมองว่าเป็นการทำ Backtest ก็ได้ คือทดสอบเพื่อให้ได้ผลลัพธ์ออกมา จากโจทย์ที่คุณตั้งไว้
3. คุณสามารถเอาผลการทำ Paper trade มาทำ Mission ได้
4. คุณสามารถเอา Model backtest ที่เราทำไว้ (ปัจจุบัน V.3) มาประกอบได้
5. โจทย์ของแต่ละวัน "ห้ามซ้ำกัน" ทุก Mission ต้องไม่เหมือนกัน แต่คุณสามารถทำ Mission relation ได้
6. ต้องมีส่วนการสรุปผลให้ผมเข้าใจด้วย save ลง db / .md ด้วยเสมอ เพื่อให้คุณและผมเข้าไปอ่านและศึกษาได้
7. เราจะสร้าง .bat 1 ตัว เพื่อปลุกคุณให้มาทำ Mission (เปิด Claude Code และยิงคำสั่ง) ดังนั้นคุณต้องสร้าง instruction ของตัวเองไว้ด้วย
8. Mission จะต่างกันกับ Model version คือ Mission สามารถ Fail ได้ ไม่จำเป็นต้องสำเร็จทุกครั้ง แต่ไม่ว่าจะ Fail / Success เราจะเอาไปต่อยอดได้เสมอ
9. หากว่า Mission สำเร็จ คุณสามารถปั้นเป็น New model version ใหม่ได้เลย (หากใกล้เคียงกับโมเดลก่อน ให้ทำเป็น V.1.1, V.1.2 ได้)
