### สิ่งที่ค้นพบ:
1. **Effective N = 1.65** -- 6 เหรียญแต่ diversification เท่ากับ 1.65 ตัว

3. **100% same direction** -- ทุก entry ไปทางเดียว ไม่เคย hedge กัน
4. **Paper trading มี 24-coin clusters** -- ถ้า signal ผิด 1 ครั้ง = ขาดทุน 24 เหรียญพร้อมกัน
5. **Cap ไม่ช่วย PnL** -- ยิ่งเข้าเยอะ expected value ยิ่งดี แต่ variance สูงขึ้นมาก
6. **WR ไม่ลดเมื่อเข้าหลายเหรียญ** (72-77%) -- signal quality ไม่เสียจาก cluster size

4. **ระบบจริง 27+ coins ต้องระวังเป็นพิเศษ** -- 24-coin cluster = เหมือนเปิด position เดียวขนาด 24x
